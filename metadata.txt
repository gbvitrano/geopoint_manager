about=GeoPoint Manager lets you create point vector layers from any data source that contains latitude/longitude fields. It supports OGR/GDAL URLs, Google Sheets (via CSV export), GeoJSON, WFS services, and existing QGIS layers. The bilingual ITA/EN interface can be switched on the fly. Data from Google Sheets is downloaded and stored persistently in a local GeoPackage with on-demand manual refresh. Compatible with both QGIS 3 (Qt5/PyQt5) and QGIS 4 (Qt6/PyQt6).
